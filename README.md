# Debaditya Bhattacharya - Portfolio Website

Welcome to my personal portfolio and research showcase website!

## About

This is a GitHub Pages-powered website built with Jekyll showcasing my professional work, projects, and academic contributions.

**Website:** [https://debbh.me](https://debbh.me)

## Features

- **Home/About:** Introduction and professional summary
- **Projects:** Showcase of key projects and work samples
- **Publications:** Academic papers and research contributions
- **Contact:** Multiple ways to get in touch

## Technologies

- **Jekyll** - Static site generator
- **GitHub Pages** - Hosting
- **Markdown** - Content format
- **HTML/CSS** - Styling and layout

## Structure

```
├── _config.yml           # Jekyll configuration
├── index.md             # Home page
├── projects.md          # Projects showcase
├── publications.md      # Research and publications
├── contact.md           # Contact information
├── assets/
│   └── images/          # Images directory
└── README.md            # This file
```

## Getting Started

1. Clone or fork this repository
2. Update the content files with your information
3. Add your profile image to `assets/images/`
4. Configure your custom domain in the CNAME file (if applicable)
5. Push to GitHub to deploy

## Customization Guide

### Update Your Information
- Edit `_config.yml` with your name, email, and social links
- Update `index.md` with your about section
- Fill in your projects in `projects.md`
- Add your publications in `publications.md`
- Update contact methods in `contact.md`

### Add Images
Place images in the `assets/images/` directory and reference them in your markdown files using:
```markdown
![Alt text](/assets/images/image-name.jpg)
```

### Change Theme
Modify the `theme:` setting in `_config.yml` to use different Jekyll themes:
- jekyll-theme-minimal (default)
- jekyll-theme-slate
- jekyll-theme-cayman
- jekyll-theme-hacker
- And more...

## Hosting on GitHub Pages

1. Create a GitHub repository named `yourusername.github.io`
2. Push this code to that repository
3. Enable GitHub Pages in repository settings
4. Your site will be live at `https://yourusername.github.io`

## Custom Domain

To use a custom domain:
1. Update the `CNAME` file with your domain
2. Configure DNS settings with your domain registrar
3. Enable HTTPS in GitHub Pages settings
