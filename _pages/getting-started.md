## Welcome to Academic Pages!

This is your Academic Pages template. Here's what you need to do to customize it:

### 1. Update Site Configuration

Edit `_config.yml` and customize:
- **Author Information**: Update your name, email, bio, and social profiles
- **Site URL**: Change `url:` to your site's URL
- **Social Links**: Add links to your GitHub, LinkedIn, Google Scholar, etc.

### 2. Add Your Content

Create content in these directories:

#### Publications (`_publications/`)
- Create a markdown file for each publication
- Include front matter with title, date, venue, paperurl, citation
- Example: `_publications/2024-01-15-my-paper.md`

#### Talks (`_talks/`)
- Add conference talks, presentations, seminars
- Include venue, date, location, and talk details
- Example: `_talks/2024-01-15-my-talk.md`

#### Teaching (`_teaching/`)
- Document courses you've taught
- Include course code, term, description, and materials
- Example: `_teaching/2024-01-15-my-course.md`

#### Portfolio (`_portfolio/`)
- Showcase projects and work samples
- Include descriptions, technologies, and links
- Example: `_portfolio/2024-01-15-my-project.md`

#### Blog Posts (`_posts/`)
- Write blog posts using standard Jekyll format
- Use YYYY-MM-DD-title.md naming convention
- Example: `_posts/2024-01-15-my-blog-post.md`

### 3. Customize Your Homepage

Edit `index.md` to add your welcome message and biography.

### 4. Update Navigation

Edit `_data/navigation.yml` to customize the top navigation menu.

### 5. Update Your CV

Edit `_pages/cv.md` to add your CV content including education, work experience, and skills.

### 6. Add Profile Picture

Replace the image at `assets/images/profile.jpg` with your own profile picture.

### 7. Install and Run Locally (Optional)

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000` to preview your site.

### 8. Deploy to GitHub Pages

Commit and push your changes to GitHub. Your site will automatically build and deploy!
